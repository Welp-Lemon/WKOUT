# WKOUT

A personal Matrix-inspired workout dashboard built with Codex.

## Features

- 💪 4-day strength and hypertrophy program
- 🟢 Matrix/Codex hologram UI
- ⏱️ 39-second core timer
- ⏲️ 60-second rest timer
- ✅ Set completion tracking
- 📈 Workout progression
- 📝 Daily workout notes
- 💾 Local storage persistence
- 👨 Dad Mode (pause workout)
- 🔊 Optional terminal sound effects
- 📱 Optimized for iPad landscape and Home Screen

## Tech

- HTML
- CSS
- Vanilla JavaScript
- LocalStorage
- No backend
- No frameworks

## Purpose

WKOUT was built as a lightweight workout companion for home gym sessions.

The goal is to provide a distraction-free experience that feels more like a mission control dashboard than a traditional fitness app.

No ads.
No subscriptions.
No accounts.
Just open it and train.
